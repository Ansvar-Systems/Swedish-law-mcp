# EU Reference Taxonomy for Swedish Law

## Overview

This document describes the patterns and formats of EU law references found in Swedish statutes, based on an analysis of 83 statutes in the Swedish Law MCP database.

## Summary Statistics

- **Total statutes analyzed:** 83
- **Statutes with EU references:** 49 (59%)
- **Statutes without EU references:** 32 (39%) - purely domestic law
- **Total EU references found:** 406
  - **Directives:** 164 (40%)
  - **Regulations:** 242 (60%)

## EU Reference Types

### 1. Directives (Direktiv)

Directives are EU legislative acts that set out goals that all EU countries must achieve, but each country can decide how to transpose them into national law.

#### 1.1 Format Patterns

Swedish statutes reference EU directives in the following patterns:

##### Pattern A: Simple directive reference
```
direktiv 2016/680
direktiv 95/46/EG
direktiv (EU) 2019/1152
```

##### Pattern B: With issuing body
```
rådets direktiv 95/46/EG
Europaparlamentets och rådets direktiv 2003/88/EG
Europaparlamentets och rådets direktiv (EU) 2019/1152
```

##### Pattern C: Full citation with date and title
```
Europaparlamentets och rådets direktiv (EU) 2016/680 av den 27 april 2016
om skydd för fysiska personer med avseende på behörig myndighets behandling
av personuppgifter för att förebygga, förhindra, utreda, avslöja eller lagföra
brott eller verkställa straffrättsliga påföljder
```

#### 1.2 Community Designations

- **EU** - European Union (2009-present)
- **EG** - Europeiska Gemenskapen (European Community, 1993-2009)
- **EEG** - Europeiska Ekonomiska Gemenskapen (European Economic Community, 1957-1993)

### 2. Regulations (Förordningar)

Regulations are directly applicable EU law that have binding legal force throughout every member state and do not require national implementing legislation.

#### 2.1 Format Patterns

##### Pattern A: Simple regulation reference
```
förordning (EU) 2016/679
förordning (EG) nr 765/2008
```

##### Pattern B: With issuing body
```
rådets förordning (EG) nr 1082/2006
Europaparlamentets och rådets förordning (EU) 2016/679
kommissionens förordning (EU) 2021/1832
kommissionens genomförandeförordning (EU) 2018/1212
kommissionens delegerade förordning (EU) 2019/815
```

##### Pattern C: Full citation
```
Europaparlamentets och rådets förordning (EU) 2016/679 av den 27 april 2016
om skydd för fysiska personer med avseende på behandling av personuppgifter
och om det fria flödet av sådana uppgifter och om upphävande av
direktiv 95/46/EG (allmän dataskyddsförordning)
```

#### 2.2 Issuing Bodies

- **Europaparlamentet och rådet** - European Parliament and Council (co-decision)
- **Rådet** - Council (Council regulations)
- **Kommissionen** - European Commission (implementing/delegated regulations)
  - **Genomförandeförordning** - Implementing regulation
  - **Delegerad förordning** - Delegated regulation

## Reference Context Types

### 1. Implementation (Genomförande)

Swedish statutes that implement EU directives often use these phrases:

```swedish
"Denna lag utgör genomförande av direktiv 2016/680"
"Genomförande av Europaparlamentets och rådets direktiv..."
"Lagen genomför direktiv..."
```

**Example:** Dataskyddslagen (2018:218) implements GDPR
```
"Denna lag kompletterar Europaparlamentets och rådets förordning (EU) 2016/679"
```

### 2. Supplementary (Komplettering)

Swedish statutes that supplement directly applicable EU regulations:

```swedish
"Lag med kompletterande bestämmelser till EU:s dataskyddsförordning"
"Bestämmelserna kompletterar förordning (EU)..."
```

### 3. Application (Tillämpning)

References to EU law that applies alongside Swedish law:

```swedish
"Vid tillämpning av förordning (EU) 2016/679..."
"I enlighet med artikel 6 i förordning (EG) nr..."
"Med stöd av artikel 9.2 i EU:s dataskyddsförordning..."
```

### 4. Consistency (Överensstämmelse)

References ensuring Swedish law aligns with EU requirements:

```swedish
"...i överensstämmelse med direktiv 2003/88/EG"
"...får inte innebära mindre förmånliga villkor än som följer av direktiv..."
```

### 5. Amendment/Repeal (Ändring/Upphävande)

References to EU acts that amend or repeal other EU acts:

```swedish
"...och om upphävande av direktiv 95/46/EG"
"...om ändring av direktiv 2014/65/EU och förordning (EU) nr 600/2014"
```

## Extraction Challenges

### 1. Article References

Many Swedish statutes reference specific articles within EU acts:

```
"artikel 6.1 c i EU:s dataskyddsförordning"
"artiklarna 13-15 i EU:s dataskyddsförordning"
"artikel 83.4, 83.5 och 83.6 i EU:s dataskyddsförordning"
```

These references require additional parsing to extract article numbers.

### 2. Indirect References

Some statutes refer to EU law indirectly:

```
"EU:s dataskyddsförordning" (without explicit citation)
"GDPR" (acronym only)
"allmän dataskyddsförordning" (informal name)
```

### 3. Amended Citations

References often indicate amendments:

```
"i lydelsen enligt Europaparlamentets och rådets förordning (EU) 2019/1020"
"i den ursprungliga lydelsen"
```

### 4. Multiple References in Single Sentence

```
"...enligt artikel 6 i förordning (EG) nr 1/2003, artikel 12.1 i rådets
förordning (EG) nr 139/2004, eller artikel 14.7 i Europaparlamentets och
rådets förordning (EU) 2022/2560..."
```

## Recommendations for Parser Design

1. **Support all three community designations:** EU, EG, EEG, Euratom
2. **Handle optional "nr" prefix** for regulations
3. **Extract issuing body** (Parliament/Council, Commission, etc.)
4. **Capture amendment context** ("i lydelsen enligt...", "i den ursprungliga lydelsen")
5. **Parse article references** within EU acts
6. **Handle abbreviations** ("EU:s dataskyddsförordning" = regulation 2016/679)
7. **Support year ranges** (2-digit vs 4-digit years: 95/46 vs 2016/679)
8. **Detect implementation keywords** ("genomförande", "komplettering", "tillämpning")
